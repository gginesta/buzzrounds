# BuzzRounds

Party games for everyone. Multiplayer browser games — no downloads, no signups.

**Live at [buzzrounds.com](https://buzzrounds.com)**

## Games

- 🎵 **Tunes** — Music trivia. Guess the year a song was released. [tunes.buzzrounds.com](https://tunes.buzzrounds.com)
- 🧠 **YDKJ** — Irreverent trivia with attitude. *Coming soon.*

## Tech

- Next.js 15 + TypeScript + Tailwind CSS
- Deployed on Vercel

## Development

```bash
npm install
npm run dev
```
